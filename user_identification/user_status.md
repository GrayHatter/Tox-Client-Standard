User Status
-----------

- ![](/badge/rec.png) A Tox user status symbol should be colour-coded according
  to the table below:

  | Userstatus           | Colour |
  |----------------------|--------|
  | Online [none]        | Green  |
  | Away                 | Yellow |
  | Busy                 | Red    |

- ![](/badge/rec.png) User status symbol shapes should be distinct from one
  another.
